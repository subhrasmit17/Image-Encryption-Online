Image Encryption Web API

Important Upload Guidelines

To keep this service free and accessible, the backend is hosted on a free-tier server. This brings a few important limitations:

  • 📌 Max image size: 1.5 MB
  • 📌 Max resolution: 1,000,000 pixels total (example: 1000×1000)
  • 📌 Key length: Max 15 characters
  • 📌 Supported formats: .jpg, .png
  • ⏳ The server may take a few seconds to respond if it's starting from sleep (only happens after long inactivity)

If the upload fails or seems slow, please double-check the image size and try again after a few seconds.
